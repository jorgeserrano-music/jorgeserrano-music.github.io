### Icon sizes
To increase icon sizes relative to their container, use `fa-xs`, `fa-sm`, `fa-lg` (33% increase)

Use literal sizes (to scale it from 1x to 10x) `fa-2x`, `fa-3x`, `fa-4x`, `fa-5x`, `fa-6x`, `fa-7x`, `fa-8x`, `fa-9x`, `fa-10x`
